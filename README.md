# Rust SQLite CLI

A simple Rust program demonstrating CRUD operations with an SQLite database.

## Dependencies
- Rust & Cargo: [Install here](https://rustup.rs/)
- SQLite

## How to Run
1. Clone this repository.
2. Run `cargo run` to execute the program.
